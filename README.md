# Projet-Domotique---ing-nierie-des-syst-mes-avec-tkinter
Application domotique en Python avec tkinter permettant de contrôler l’éclairage et la température de chaque pièce. L’interface propose boutons, sliders et messages de feedback. Le projet applique les principes ergonomiques vus en cours pour offrir une interaction simple, cohérente et intuitive
